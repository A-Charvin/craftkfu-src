#craftkfu

> Craftkfu is an unofficial tool without any link with Ankama, editor of WAKFU.

A simple, only client-side tool, helping with crafting on Wakfu.

#use

git clone https://github.com/Vertylo/craftkfu-src.git

``` bash
git clone https://github.com/Vertylo/craftkfu-src.git

cd craftkfu-src

npm i
npm run dev
```
